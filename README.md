# Home Assistant Apps

Personal Home Assistant add-on repository.

## Installation

1. In Home Assistant, go to **Settings → Apps**
2. Click the menu (⋮) → **Repositories**
3. Add: `https://github.com/bjornwein/ha-apps`
4. Refresh — apps from this repository will appear

## Apps

### [GrillSense Thermometer](grillsense/)

Monitor your GrillSense / Dangrill WiFi BBQ thermometer.
6 temperature probes, alarm control, fully local operation.
